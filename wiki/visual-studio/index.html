Tools - Visual Studio

Wiki Tools

地球上最好用的 C++ 开发环境，没有之一。现在也有免费的社区版了，可惜以前密集写 C++ 代码的时候没赶上。

快捷键

功能快捷键
快速打开 anythingC-,
当前打开文件C-M-Down
切换头/源文件C-ko
返回C–
前进C-S–
跳转到定义F12
在新窗口展示定义M-F12
跳转到声明C-M-F12
上/下移动行M-Up/Down
在上面插入行C-Enter
在下面插入行C-S-Enter
折叠/展开当前代码段C-mm
注释C-kc
取消注释C-ku
选中单词大写C-S-u
选中单词小写C-u
调试F5
执行C-F5
设置断点F9
步过F10
步入F11

插件

  • Go To Definition

    Make ctrl+click perform a “Go To Definition” on the identifier under the cursor.

  • Visual Assist

    Productivity tool for C/C++ and C# that improves IDE features related to navigation, refactoring, code generation, and coding assistance.
